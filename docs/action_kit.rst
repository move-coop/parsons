ActionKit
=========

********
Overview
********

The ActionKit class allows you to interact with an `ActionKit <https://actionkit.com/>`_.

**********
Quickstart
**********

**Retrieve a User**

.. code-block:: python

  from parsons import ActionKit
  ak = ActionKit()
  user_data = ak.get_user(123)
