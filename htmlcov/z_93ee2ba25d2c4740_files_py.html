375class TempFile:

381 is called explicitly, the temporary file is removed from disk. 

382 

383 Unlike NamedTemporaryFile from the Python standard library, this class does NOT represent 

384 an open file handle to the file. It simply represents a file on disk. This class was 

385 written to workaround the fact that on Windows, NamedTemporaryFile opens the file with an 

386 exclusive read lock, which means that no one else can open the file for reading. 

387 

388 Since Parsons hands out the temporary file's path and not the file handle, users must be able 

389 to open the file, but that is impossible as long as NamedTemporaryFile holds onto the open 

390 file handle with its exclusive read lock. So we wrote, TempFile to not hold onto the open 

391 file handle. 

392 

393 `Args:` 

394 suffix: str 

395 The suffix to give the file path in order to advertise the file/mime type of the file. 

396 """
